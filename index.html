HTML

HTML, or HyperText Markup Language, is the standard markup language used to create and design web pages. It provides the structure and content of a webpage through a system of tags and elements.

CSS

CSS, or Cascading Style Sheets, is a styling language used to enhance the appearance and layout of HTML elements on a webpage. It allows developers to define colors, fonts, spacing, and other visual properties, ensuring consistency and aesthetic appeal across the site.

Bootstrap

Bootstrap is a widely-used framework for building responsive websites and applications. It offers pre-designed components and styles, streamlining development and ensuring a modern, consistent look and feel.
